# enunciado


Desenvolva um sistema para gestão de cardápios e refeições em um restaurante universitário. O
sistema deve permitir o cadastro de refeições, ingredientes, cardápios semanais, estudantes e registros
de consumo. Cada refeição deve possuir nome, tipo, ingredientes e informações sobre restrições
alimentares. O cardápio deve organizar refeições por dia e turno. O sistema deve permitir consultar
cardápio, registrar consumo, filtrar refeições por restrição alimentar e impedir cadastro duplicado de
refeição no mesmo dia e turno. A solução deve respeitar toda a estrutura de camadas definida no PRD
